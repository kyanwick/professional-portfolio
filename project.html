Teller's Gambit - A Bank Teller Simulator (In Development)

Inspired by Papers, Please, this project features pixel art, day-to-day choices, and a 70s-themed soundtrack to create an immersive, retro experience. Developed in Unity with C#.

Disk Usage Utility

C++ Visual Studio Code

This project implements a custom disk usage (du) utility inspired by Unix, aimed at calculating and displaying disk space usage across files and directories. The program collects information from the command line, analyzes disk usage by accessing file sizes and directory structures, and generates a report based on user-defined options.
